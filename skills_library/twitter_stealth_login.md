# 技能名称: 推特：矩阵账号免密登录
# 描述: 自动登录推特账号，优先尝试注入 Cookie 静默登录，如果失效则回退到 Vault 金库物理登录。
# 所属分类: 推特（X）自动化套件

## 参数
- {{twitter_account_id}}: 账号 ID
- {{identity_token}}: 用于静默登录的 Identity Token（可选）

## 步骤
1. 导航到 https://x.com/i/flow/login
2. 检查当前是否已由于 Cookie 保持登录态。如果首页正常加载且显示个人头像，则跳过后续步骤。
3. 如果未登录，首先尝试从 Vault 加密库中读取并注入 {{identity_token}} 对应的 Session Cookies（L3 基础设施能力）。
4. 刷新页面，验证静默登录是否成功。
5. 如果静默登录失效，则定位输入框，调用 Vault 提取明文账号 {{twitter_account_id}}。
6. 执行物理拟人输入：输入账号并按回车（L2 键盘模拟）。
7. 定位并输入密码（L3 安全提取），随机停顿 1-2 秒。
8. 模拟人类物理点击“Log in”按钮（贝塞尔曲线轨迹）。
9. 如果触发 2FA（两步验证），则进入“等待人类接管”状态或自动捕获验证码输入。
10. 登录完成后，自动同步最新的 Cookies 到本地金库，确保下次可以静默启动。
