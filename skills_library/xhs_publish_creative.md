# 技能名称: 小红书：笔记创意发布
# 描述: 自动生成文案并模拟人类点击上传流程，发布具有吸引力的小红书笔记。
# 所属分类: 小红书自动化套件

## 参数
- {{topic}}: 笔记主题（如：AI 绘画教程、周末露营分享）
- {{image_prompt}}: 给大模型生成的图片描述词（用于后续可能的图片生成或选择）
- {{hashtags}}: 目标标签列表

## 步骤
1. 导航到 https://www.xiaohongshu.com/publish
2. 检查页面是否处于发布模式。如果是，定位“上传图片/视频”区域。
3. 模拟物理点击“上传”按钮（高斯随机落点），触发系统文件选择（由底层物理模拟器执行文件路径注入）。
4. 在“标题”框中，输入由 LLM 基于 {{topic}} 生成的标题（模拟人类打字速度）。
5. 在“正文内容”框中，输入由 LLM 基于 {{topic}} 和 {{hashtags}} 生成的具有小红书风格的文案（表情符号引用、语气亲和）。
6. 定位“发布”按钮，但在点击前随机停留 3-5 秒，模拟人类在最后确认内容。
7. 执行贝塞尔轨迹物理拟人点击“发布”按钮。
8. 等待进度条完成，捕获并保存发布成功的确认页面截图到 Artifacts 库。
9. 如果出现内容审核提示，自动记录日志并标记为“审核中”。
10. 返回个人主页，检查新发布的笔记是否可见。
