# 技能名称: 小红书：账号安全登录
# 描述: 自动检测小红书登录状态，并在必要时从 Vault 金库中提取凭证进行物理模拟登录。
# 所属分类: 小红书自动化套件

## 参数
- {{account_id}}: 执行登录的账号 ID（对应 Vault 中的标识）

## 步骤
1. 导航到 https://www.xiaohongshu.com
2. 检查页面右上角。如果看到“登录”按钮，则说明需要执行登录流程。
3. 点击“登录”按钮，弹出登录模态框。
4. 选择“帐号密码登录”选项（如果默认是扫码）。
5. 调用 Vault 金库，根据 {{account_id}} 获取对应的手机号/邮箱和密码。
6. 执行物理拟人输入：先输入账号，随机停顿 0.5 秒，再输入密码。
7. 模拟物理点击“登录”按钮（贝塞尔曲线轨迹）。
8. 如果弹出滑块验证码，则暂停执行并发出“人类接管 (Takeover)”请求，或尝试调用识别服务。
9. 登录成功后，等待 3 秒加载首页，并验证头像是否出现。
10. 将最新的 Session Cookies 自动回写到本地安全存储，以便下次静默登录。
