#### 阶段 3：更新机制（持续运营）
**自动更新方案**（Electron autoUpdater，顶级标准）

```ts
// main.ts
import { autoUpdater } from 'electron-updater';

autoUpdater.on('update-available', () => {
  dialog.showMessageBox({ message: '新版本可用，正在下载...' });
});

autoUpdater.on('update-downloaded', () => {
  dialog.showMessageBox({ message: '更新完成，重启应用？' }).then(() => autoUpdater.quitAndInstall());
});

app.on('ready', () => autoUpdater.checkForUpdatesAndNotify());
```

**更新频率**：每月 1 次大更新，每周小修复。

**版本策略**：
- Free 用户：强制更新（防旧版漏洞）。
